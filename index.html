What I Do ?


As A Full Stack Developer, I'm Involved In Building Both The Front End (Client-Side) And Back End (Server-Side) Parts Of A Web Application Or Software.


Javascript


JavaScript Is A Versatile And Widely Used Programming Language Primarily Employed For Building Dynamic And Interactive Content On Websites. It Is A Client-Side Scripting Language, Meaning It Runs In A User's Web Browser Rather Than On A Server. JavaScript Allows Developers To Enhance The User Experience By Creating Responsive And Interactive Elements Within Web Pages.


Github


GitHub Is A Web-Based Platform That Serves As A Collaborative Development Platform For Software Projects. It Is Primarily A Version Control System That Enables Multiple Developers To Work On A Project Simultaneously Without Conflicting With Each Other's Changes. Developed Using Git, GitHub Provides A Centralized Hub For Managing And Tracking Changes To Source Code.


Deployment


Deployment, In The Context Of Software Development, Refers To The Process Of Making A Software Application Or System Available For Use. It Involves Taking The Codebase, Configuration Settings, And Other Necessary Assets Developed During The Software Development Lifecycle And Placing Them In A Production Environment. This Production Environment Is The Actual Server Or Hosting Infrastructure Where The Application Will Run And Be Accessed By End-Users.
